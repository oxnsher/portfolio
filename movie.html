Movie Reviews Website



This website allows users to browse available movies and read reviews submitted by other users. It includes a search function that enables users to find movies using keywords, and a filter feature that lets them sort movies based on language.

Guests have the option to register for an account to become members. Once logged in, members can update, delete, or log out of their accounts at any time. Additionally, they can submit reviews for movies, as well as edit or delete their own reviews, offering a personalized and interactive experience.

This project demonstrates my ability to build a dynamic and user-centric platform with essential CRUD functionalities and responsive design principles.

Software: NetBeans IDE
Programming Language: HTML, PHP, CSS, MySQL and Bootstrap
