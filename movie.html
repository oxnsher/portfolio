Coded Website



Movie Reviews Website

This website was coded using HTML, PHP, CSS, MySQL and Bootstrap on NetBeans IDE. It allows the user to view available movies and reviews made by other users. Users can also search for movies using keywords and filter the movies based on its language. Guests can register for an account to become a member. After logging in, members can update, delete or log out of their account. Members can also leave reviews for movies and edit or delete their own reviews.
