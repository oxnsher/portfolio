Movie Collection Application



This app allows users to add movies along with key details such as title, genre, and rating. All movies entered by the user are collated and displayed in a clean, list format, providing a clear overview of their personal movie collection.

Users can easily edit or delete movie entries, making the app dynamic and user-friendly. To enhance the experience, the app also includes a filtering feature that automatically displays only movies rated PG13 and below, helping users quickly find age-appropriate content.

This project demonstrates my ability to implement CRUD functionalities, manage dynamic data, and design a simple yet effective user interface for organizing and interacting with information.

Software: Android Studio
Programming Language: Java
