Movie List Mobile Application

This app allows users to insert movies and the movie details (e.g genre, rating, title). It then collates all the movies that the user has inserted and shows it to the user in a list form. The user is also able to edit and delete movies. It also filters out all PG13 and below movies for the user.

Software: Android Studio
Programming Language: Java
